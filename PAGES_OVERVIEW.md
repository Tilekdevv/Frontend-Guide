## 🚀 Как добавить новую страницу

### 1. Создайте файл страницы

```bash
src/pages/YourPage.jsx
```

### 2. Добавьте роут в App.jsx

```jsx
<Route path="/category/your-slug" element={<YourPage />} />
```

### 3. Обновите slug в categories.js

```js
{
  id: X,
  slug: 'your-slug',
  title: 'Your Title',
  // ...
}
```

### 4. Используйте общие компоненты

- Navbar
- Section
- ContentCard
- InfoBox
- CodeExample

### Breakpoints

- **Mobile:** 320px - 767px
- **Tablet:** 768px - 1023px
- **Desktop:** 1024px+

### Адаптация

- Grid: 3 → 2 → 1 колонка
- Sidebar: скрывается на mobile
- Таблицы: горизонтальный скролл
- Шрифты: уменьшаются на mobile
