# Функционал страницы "IT и путь разработчика"

## ✨ Реализованные возможности

### 📱 Sidebar Navigation

- Фиксированная боковая панель с навигацией по секциям
- Автоматическое выделение активной секции при скролле
- Плавная прокрутка к секции при клике
- Иконки для каждой секции
- Адаптивный дизайн (скрывается на мобильных)

### 📚 Контент страницы

#### 1. Что такое IT

- Простое объяснение IT и Frontend разработки
- Аналогия с рестораном для понимания
- Карточки основных технологий (HTML, CSS, JavaScript)
- Реальный пример с Instagram

#### 2. Как развивается разработчик

- Карьерные этапы: Junior → Middle → Senior
- Описание навыков для каждого уровня
- Типичные ошибки новичков с решениями
- Советы для студентов

#### 3. Почему важно учиться постоянно

- Объяснение быстрых изменений в индустрии
- Реальный пример эволюции технологий (jQuery → React)
- Карточки областей для изучения
- Важность английского языка

#### 4. Реальная работа Frontend разработчика

- Типичный рабочий день с расписанием
- Реальный пример задачи (кнопка "Поделиться")
- Описание команды (PM, Designer, Backend, QA)
- Важность коммуникации

### 🎨 UI Компоненты

#### InfoCard

Цветные карточки с иконками для важной информации:

- 🔵 Синий - "Простыми словами"
- 🟣 Фиолетовый - "Реальный пример"
- 🟢 Зелёный - "Совет студентам"
- 🟠 Оранжевый - "Важно знать"

#### TechCard

Карточки технологий с градиентами и hover эффектами

#### CareerStage

Этапы карьеры с описанием и списком навыков

#### MistakeItem

Ошибки новичков с иконками ❌ и решениями ✅

#### LearningCard

Области для изучения со списками

#### DayActivity

Расписание рабочего дня с временем

#### TeamMemberCard

Описание членов команды

### 🎭 Анимации

- Fade in при появлении секций
- Hover эффекты на карточках
- Плавная прокрутка
- Smooth transitions

### 📱 Адаптивность

- Desktop: sidebar + контент
- Tablet: только контент, полная ширина
- Mobile: оптимизированная вёрстка

### 🎯 Навигация

- Кнопка "Назад к категориям"
- Автоматическое отслеживание позиции скролла
- Sticky sidebar
- CTA в конце страницы

## 🚀 Как использовать

1. Откройте главную страницу
2. Кликните на карточку "IT и путь разработчика"
3. Используйте sidebar для навигации по секциям
4. Читайте контент, изучайте примеры
5. Следуйте советам и избегайте ошибок новичков
